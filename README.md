# Python AI Engineering Portfolio

This repository is a collection of **small Python projects and code examples** created as part of my journey to become an **AI Engineer**.

The goal of this project is to practice and demonstrate different Python concepts through small, practical projects. Each example helps build the skills needed for working with **AI, data processing, and real-world applications**.

---

## Project Purpose

This project serves as:

* A learning path toward becoming an **AI Engineer**
* A collection of **small Python projects**
* A personal **code portfolio**
* A place to store reusable Python examples

The repository will continue to grow as I learn new technologies and build more projects related to **AI and Python development**.

---

## How to Run the Project

### 1. Install dependencies

```bash
pip install streamlit
```

### 2. Run the application

```bash
streamlit run app.py
```

After running the command, the Streamlit app will start and open in your browser.
